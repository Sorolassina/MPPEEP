📂 Charger une Fiche Technique

📋 Processus de chargement :
  1. Analyse automatique : Le système analyse le fichier et détecte la structure
  2. Reconnaissance des colonnes : Identification des colonnes clés (Code, Libellé, Montants)
  3. Hiérarchisation : Organisation en Actions → Services → Activités → Lignes
  4. Validation : Vérification des données et création du tableau
📝 Format requis pour le fichier Excel :
🏗️ Structure hiérarchique (ordre exact) :
  1. Nature de Dépenses : BIENS ET SERVICES, PERSONNEL, INVESTISSEMENT, TRANSFERTS
  2. Actions : Commencent par Action : ou - Action :
  3. Services Bénéficiaires : Commencent par Service Bénéficiaire : ou - Service Bénéficiaire :
  4. Activités : Commencent par Activité : ou - Activité :
  5. Lignes budgétaires : Commencent par un numéro de compte (ex: 601100, 621200)
📊 Colonnes attendues :
  • CODE ET LIBELLE : Identification et description
  • BUDGET VOTÉ N : Budget voté année précédente (A)
  • BUDGET ACTUEL N : Budget actuel année précédente (B)
  • ENVELOPPE N+1 : Enveloppe demandée année suivante (C)
  • COMPLÉMENT SOLLICITÉ : Demande de complément (D)
  • BUDGET SOUHAITÉ : Budget souhaité total (E=C+D)
  • ENGAGEMENT ÉTAT : Engagement de l'État (F)
  • AUTRE COMPLÉMENT : Autre source de financement (G)
  • PROJET DE BUDGET N+1 : Projet final (H=C+F+G)
  • JUSTIFICATIFS : Notes explicatives
