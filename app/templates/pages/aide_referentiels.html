📚 Système de Gestion des Référentiels

Les référentiels constituent la base de données de référence de l'application. Ils définissent la structure organisationnelle (programmes, directions, services) et les classifications (grades, catégories) utilisées dans tous les modules.

🏢 Structure Hiérarchique

📊 Programme (Niveau 1)
   └─ 📁 Direction (Niveau 2)
      └─ 🏪 Service (Niveau 3)
         └─ 👤 Agents

Exemple concret :

📊 Programme: ADMIN (Administration Générale)
   └─ 📁 Direction: DAAF (Direction des Affaires Administratives et Financières)
      └─ 🏪 Service: SBUD (Service Budget)
         ├─ 👤 Chef de service: Lassina SORO
         └─ 👤 Agents du service...

📊 Gérer les Programmes

➕ Créer un programme

  1. Cliquer sur "➕ Nouveau Programme"
  2. Remplir le formulaire :
    • Code : Identifiant court (ex: ADMIN, EDUC, SANTE)
    • Libellé : Nom complet du programme
    • Description : Détails optionnels
    • Responsable : Agent responsable du programme (optionnel)
  3. Enregistrer

✏️ Modifier / 🗑️ Supprimer

Utiliser les actions dans la colonne "Actions" du tableau. La suppression est un soft delete : le programme est désactivé mais conservé en base.

📁 Gérer les Directions

Les directions sont rattachées à un programme parent.

  1. Cliquer sur "➕ Nouvelle Direction"
  2. Sélectionner le programme de rattachement
  3. Remplir :
    • Code (ex: DAAF, DRH, DTIC)
    • Libellé (nom complet)
    • Description
  4. Enregistrer

💡 Astuce : Le filtrage par programme permet de visualiser la structure hiérarchique et de vérifier la cohérence organisationnelle.

🏪 Gérer les Services

Les services sont rattachés à une direction parente et peuvent avoir un chef de service.

  1. Cliquer sur "➕ Nouveau Service"
  2. Sélectionner la direction de rattachement
  3. Remplir :
    • Code (ex: SBUD, SCPT, SAPV)
    • Libellé (nom complet)
    • Description
    • Chef de service : Sélectionner un agent (optionnel)
  4. Enregistrer

👤 Attribuer un Chef de Service

Le chef de service peut être assigné lors de la création ou modification du service. Sélectionnez un agent dans la liste déroulante. Seuls les agents actifs sont affichés.

🎖️ Gérer les Grades

Les grades définissent les classifications de la fonction publique.

Catégories de grades

Catégorie A

Cadres supérieurs

Catégorie B

Cadres intermédiaires

Catégorie C

Personnel d'exécution

  1. Cliquer sur "➕ Nouveau Grade"
  2. Remplir :
    • Code (ex: A1, B2, C3)
    • Libellé (nom du grade)
    • Catégorie (A, B, C, Hors catégorie)
    • Indices (min/max)
  3. Enregistrer

🔄 Impact sur les Autres Modules

Les référentiels sont utilisés dans tous les modules de l'application :

  • Personnel : Affectation des agents (programme, direction, service, grade)
  • RH : Workflows personnalisés, hiérarchie de validation
  • Budget : Structure budgétaire, fiches techniques
  • Stock : Affectation des articles et demandes
  • Performance : Objectifs par programme/direction

⚠️ Attention : Modifier ou supprimer un référentiel peut impacter tous les modules. Vérifiez les dépendances avant toute modification importante.
