👥 Système de Gestion du Personnel

Le module Personnel centralise toutes les informations relatives aux agents : état civil, affectation, carrière, documents et évaluations.

➕ Créer un Agent

📝 Informations obligatoires

Champs obligatoires :

  • Matricule (unique)
  • Nom et Prénom
  • Date de recrutement
  • Date de prise de service
  • Affectation complète : Grade, Programme, Direction, Service, Fonction

🏢 Sections du formulaire

🆔 Identification

Matricule, CNI, Passeport

👤 État Civil

Nom, prénom, date de naissance, situation familiale, etc.

📞 Coordonnées

Emails, téléphones, adresse

💼 Carrière

Dates de recrutement, prise de service, retraite prévue

🏢 Affectation Actuelle (Obligatoire)

Grade, Programme, Direction, Service, Fonction

🏖️ Congés

Solde annuel, congés de l'année en cours

📸 Photo

Photo de profil (optionnelle, formats: JPG, PNG)

👤➕ Créer un Compte Utilisateur pour un Agent

Pour permettre à un agent de se connecter à l'application et créer des demandes RH :

  1. Dans la liste des agents, identifier celui sans compte (icône 👤➕ bleue)
  2. Cliquer sur l'icône pour ouvrir le modal
  3. Saisir l'email de connexion (pré-rempli avec suggestion)
  4. Définir ou générer un mot de passe sécurisé
  5. Confirmer la création
  6. Noter le mot de passe affiché et le communiquer à l'agent

✅ Résultat : L'agent pourra se connecter et créer des demandes RH. Son identité sera automatiquement renseignée dans les formulaires.

📄 Gérer les Documents d'un Agent

Chaque agent peut avoir plusieurs documents attachés :

  • CNI : Carte Nationale d'Identité
  • Diplômes : Attestations de formation
  • Contrats : Contrat de travail, avenants
  • Actes : Actes administratifs (nomination, affectation, etc.)
  • Autres : Documents divers

Pour ajouter un document, aller dans la fiche détaillée de l'agent et utiliser la section "Documents".

📚 Historique de Carrière

L'historique permet de tracer toute la carrière de l'agent :

  • Avancements : Changements de grade et échelon
  • Mutations : Changements de service ou direction
  • Nominations : Nouvelles fonctions
  • Formations : Formations suivies

💡 Astuce : L'historique est automatiquement mis à jour lors des modifications de grade, service ou fonction.

🔍 Recherche et Filtres

Options de recherche :

  • Recherche globale : Par matricule, nom, prénom ou email
  • Filtre par service : Afficher uniquement les agents d'un service
  • Filtre par grade : Catégorie A, B, C
  • Filtre par statut : Actifs / Inactifs

La recherche est instantanée (minimum 2 caractères) et affiche les résultats en temps réel.

✨ Bonnes Pratiques

  • ✅ Vérifier l'unicité du matricule avant création
  • ✅ Compléter l'affectation obligatoire (grade, programme, direction, service, fonction)
  • ✅ Ajouter une photo pour faciliter l'identification
  • ✅ Créer un compte utilisateur pour les agents qui doivent accéder à l'application
  • ✅ Maintenir les données à jour (mutations, avancements)
  • ✅ Archiver les documents importants (contrats, diplômes, actes)
  • ✅ Désactiver plutôt que supprimer pour conserver l'historique
