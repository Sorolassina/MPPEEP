💰 Système de Gestion Budgétaire

Le module Budget permet de gérer l'ensemble du cycle budgétaire : élaboration des fiches techniques, suivi de l'exécution, intégration SIGOBE et rapports financiers.

📊 Fiches Techniques Budgétaires

Structure d'une fiche

  • Programme : Niveau stratégique (ex: ADMIN, EDUC)
  • Actions : Déclinaison du programme
  • Lignes budgétaires : Détail des dépenses par nature
  • Montants : AE (Autorisations d'Engagement) et CP (Crédits de Paiement)

➕ Créer une fiche technique

  1. Cliquer sur "➕ Nouvelle Fiche"
  2. Sélectionner :
    • Année budgétaire
    • Programme
    • Phase (Élaboration, Exécution, Clôture)
  3. Ajouter des lignes budgétaires avec les montants
  4. Joindre des documents justificatifs (optionnel)
  5. Enregistrer comme brouillon ou soumettre pour validation

💡 Astuce : Utilisez le filtre par année et programme pour retrouver rapidement vos fiches.

🔄 Intégration SIGOBE

SIGOBE (Système Intégré de Gestion des Opérations Budgétaires de l'État) est le système national de gestion budgétaire. L'intégration permet d'importer et analyser les données d'exécution.

📤 Charger un fichier SIGOBE

  1. Aller dans "Budget → SIGOBE"
  2. Cliquer sur "📤 Charger un fichier"
  3. Sélectionner :
    • Année d'exécution
    • Trimestre (optionnel)
    • Fichier Excel (.xlsx ou .xls)
  4. Le système analyse automatiquement le fichier
  5. Vérifier la prévisualisation (colonnes, programmes, actions)
  6. Confirmer l'import

📊 Actions disponibles

  • 📋 Voir la table : Affichage complet des données importées
  • 📊 Exporter Excel : Télécharger les données au format Excel
  • 📈 Voir les KPIs : Statistiques et indicateurs budgétaires
  • 🗑️ Supprimer : Effacer un chargement (avec confirmation)

⚠️ Format requis : Le fichier Excel doit respecter le format SIGOBE standard avec les colonnes obligatoires (Programmes, Actions, Rprog, Type_depense, etc.).

📈 Rapports et Exports

Types de rapports disponibles

📊 Rapport d'Exécution

Suivi détaillé de l'exécution budgétaire par programme, avec taux de consommation.

💹 Analyse de Performance

Comparaison prévisions vs réalisations, identification des écarts.

📉 Tableau de Bord

Vue synthétique avec KPIs principaux et graphiques.

✨ Bonnes Pratiques

  • ✅ Vérifier la cohérence des montants avant soumission
  • ✅ Joindre les documents justificatifs aux fiches importantes
  • ✅ Valider régulièrement les données SIGOBE importées
  • ✅ Exporter les rapports pour archivage et audit
  • ✅ Respecter les phases budgétaires (Élaboration → Exécution → Clôture)
  • ✅ Maintenir la structure hiérarchique à jour dans les référentiels
